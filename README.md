# **Descrição do jogo**
* **Género:** Endless Runner
* **Objetivo:** Conseguir um valor alto de pontuação e bater o próprio recorde.

### **Funcionalidades implementadas:**
* **Menus:** Menu Principal e Menu de Pausa.
* **Estado de Jogo:** Ecrã de Game Over.
* **Progressão:** Dificuldade dinâmica consoante a pontuação.
* **Configurações:** Menu de Definições.
* **Economia:** Loja (compras de itens que facilitam a jogabilidade do jogador).
* **Acessibilidade:** Suporte multilíngue (Português, Espanhol e Inglês).
* **Inimigos:** Variedade de obstáculos (Diferentes Cactos e Corvos).
* **Arte Própria:** Sprites do cacto e do jogador desenvolvidos manualmente.
* **Multimédia:** Áudios e outros sprites integrados.

### **Jogabilidade / Controlos:**
* **Jump (Saltar):** Tecla `W` ou `Seta para cima`
* **Slide (Deslizar):** Tecla `S` ou `Seta para baixo`
* **Usar item Deadeye (Deslizar):** Tecla `F`
