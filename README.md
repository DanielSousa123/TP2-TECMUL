# **Como Executar**

# **Descrição do jogo**
**Género:** Endless Runner
**Objetivo:** Conseguir um valor alto de pontuação e bater o próprio recorde.
**Funcionalidades implementadas**
**-Menu Principal/Menu Pausa**
**-Game Over**
**-Dificuldade consoante a pontuação**
**-Definições**
**-Loja**(compras de items que facilitam a jogabilidade do player)
**-Suporte multilíngue**(português,espanhol e inglês)
**-Variedade Obstáculos**(Diferentes Cactos e Corvos)
**Sprites do cacto e player desenvolvidos manualmente**
**-Multimédia**(audios e outros sprites).

**Jogabilidade / controlos** 
Jump/Saltar- Tecla W ou Seta para cima
Deslizar/Slide- Tecla S ou Seta para baixo
